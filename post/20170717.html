[Octave] パッケージのインストールとロード

作成日: 2017/07/17

Matlabが使える環境だとMatlabを使えば良いのですが，使えない場合にOctaveを使うことがあります． ただしMatlabでデファルトで使える関数が使えなかったり，パッケージが足りなかったりするので，適宜調べて補う必要があります． 今回，たまたま使っていたプログラムでkmeansが動かなかったので，プロンプトを参考に解決しました．

pkg install -forge io  # Verが古いのでアップデート
pkg install -forge statistics
pkg load statistics

また.cファイルを読み込むために，ubuntuでliboctave-devをインストールしました．

aptitude install liboctave-dev
